# Limits

- You can only have a maximum of 500 custom detection rules configured at any time.  
- We allow maximum 20 tags to be configured for a detection rule.
- You can schedule continuous query in between 5-60 minutes on a multiple of 5.
- We do not allow user to update default detection rules other than enable/disable as per needs. We advise to clone the rule if required.
