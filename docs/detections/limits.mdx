# Limits

-   Maximum of 500 custom detections.
-   Maximum of 20 tags per detection.
-   Detection frequency range is from 5-60 minutes with options being multiples of 5.
-   Default detections are not editable. They can only be enabled / disabled. In order to make edits, please clone the detection.
